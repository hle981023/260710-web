# Red Introduction Button Design

## Goal

Make the `자기소개로...` link in `index.html` read visually as a red button.

## Design

- Change only the `.next-link` color declarations in `index.html`.
- Use `#e60012` for the background and white for the text.
- Keep the existing black border, spacing, typography, position, visibility timing, destination, and click behavior unchanged.
- Do not change buttons or links on other pages.

## Validation

- Verify the `.next-link` CSS contains the approved background and text colors.
- Run the existing static-site validation to confirm that all local HTML references still resolve and no file exceeds the upload-size threshold.
- Confirm the git diff contains only the intended button color change and this design record.
